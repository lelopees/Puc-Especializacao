PCA tecnica que usa algebra linear para construir retas no planos dos dados de forma que os dados de diferentes classes fiquem separados
é indicado quando não se sabe quais features deseja-se exluir, indicando o não conhecimento da base de dados
é um metodo linear, limitando sua atuação em dados não lineares. Busca por uma projeção linear a partir dos dados originais em um outro plano com uma dimensionalidade menor, exemplo: em um dataset de vinhos, no qual possui 4 features (teor alcoolico, coloração do vinho, acidez e tipo da uva) pode-se reduzir as dimensões, armazenando teor alcoolico e coloração do vinho em um unico valor.

LDA usada para quando se conhece mais os dados , similar ao PCA, porem permite analise mais correta em dados não lineares

t-SNE técnica que usa estatistica para comparar as distribuições dos dados e inferir se são da mesma classe ou não, permitindo separar os dados de forma mais clara.
procura um balanceamento onde as duas distribuições apresentem a maior similaridade. Essa ténica possui limitações na quantidade maxima de dimensões, sendo recomendadas até 50 dimensões, caso seja maior aplicada PCA para reduzir a um numero menor que 50 antes de usar t-sne.
A métrica divergencia KL representa o erro, que pode haver na transformação dos dados, quanto menor o valor, melhor o modelo.
